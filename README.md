# Тренажер Церковных Тропарей

## Описание проекта

Современное веб-приложение для изучения и запоминания православных тропарей воскресных и праздничных дней. Приложение сочетает традиционную церковную эстетику с современным дизайном и удобным интерфейсом.

## Особенности приложения

### 🎨 Дизайн и интерфейс
- **Современная интерпретация церковной эстетики** - сочетание традиционных цветов (бордовый, золотой) с современным минимализмом
- **Адаптивный дизайн** - идеально работает на десктопах, планшетах и мобильных устройствах
- **Плавные анимации и переходы** - приятный пользовательский опыт
- **Интуитивная навигация** - понятный интерфейс без перегрузки элементами

### 📚 Контент
- **Все 8 воскресных тропарей** - полный набор по гласам
- **12 двунадесятых праздников** - главные христианские праздники
- **Индивидуальные советы по запоминанию** - для каждого тропаря разработаны специальные рекомендации
- **Аудио сопровождение** - возможность прослушивания тропарей (требуется загрузка аудиофайлов)

### 🎯 Механика обучения

#### Основной режим - постепенное исчезновение слов
- **Полный текст** - для первоначального знакомства
- **20% скрытия** - легкий уровень для начала
- **50% скрытия** - средний уровень сложности  
- **80% скрытия** - сложный уровень для проверки знаний
- **Только первые буквы** - максимальный уровень сложности

#### Дополнительные возможности
- **Своя сложность** - регулировка процента скрытых слов с помощью слайдера
- **Перемешать слова** - новая генерация скрытых слов при текущем уровне сложности
- **Показать текст** - мгновенное отображение полного текста

### 🔊 Аудио функции
- **Встроенный аудиоплеер** с удобными элементами управления
- **Прогресс воспроизведения** - визуализация текущего времени
- **Предпрослушивание** - быстрый просмотр аудио при выборе тропаря
- **Поддержка Google Drive** - загрузка аудиофайлов по прямым ссылкам

## Использование

### Базовый сценарий использования
1. **Выбор тропаря** - переходите по категориям или выбирайте случайный тропарь
2. **Прослушивание** - при необходимости прослушайте аудио для правильного произношения
3. **Изучение текста** - начинайте с полного текста для знакомства
4. **Постепенное запоминание** - увеличивайте уровень скрытия слов по мере запоминания
5. **Проверка знаний** - используйте уровень 80% для самопроверки

### Рекомендации по запоминанию
- Начинайте с **полного текста** (0%)
- Переходите к **20% скрытия** когда почувствуете уверенность
- Используйте **50% скрытия** для активного запоминания
- Проверяйте себя на **80% скрытия** перед завершением
- Завершайте **только первыми буквами** для полного закрепления

### Советы по работе с аудио
Для работы аудио необходимо:
1. Загрузить аудиофайлы на Google Drive
2. Получить ID файлов и заменить placeholder'ы в коде
3. Убедиться, что файлы доступны для просмотра по прямой ссылке
